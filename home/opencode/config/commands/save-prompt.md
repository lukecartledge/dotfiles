Save a reusable prompt to the Obsidian vault.

1. Create a new file at ~/notes/dev-vault/opencode/prompts/$ARGUMENTS.md
2. Use this frontmatter:
---
type: prompt
created: {{date}}
tags: []
source-session:
status: untested
---
3. Write the prompt exactly as it should be reused
4. Add a ## Context section explaining when to use it
5. Add a ## Example section with a sample input and output
